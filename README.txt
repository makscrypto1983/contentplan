# Генератор Контент-Плана

Этот программный инструмент предназначен для генерации структуры папок и файлов, которые могут быть использованы для планирования контента на год. Программа создает каталоги с названиями месяцев и дней недели на русском языке и внутри каждого дня создает файл "Статья.txt". 

## Инструкции по использованию

1. Запустите программу "Генератор Контент-Плана" (exe-файл).

2. После запуска, на экране появится главное окно с описанием программы и двумя кнопками.

3. Нажмите кнопку "Сгенерировать контент план" для создания структуры папок и файлов. В результате, в папке, где находится исполняемый файл программы, будет создан каталог "Контент план" с подкаталогами, соответствующими месяцам и дням недели.

4. После завершения генерации, появится уведомление о завершении процесса.

5. Вы можете использовать созданные папки и файлы для организации своего контент-плана.

6. По завершению работы, нажмите кнопку "Выход", чтобы закрыть программу.

## Требования

Программа работает на операционных системах Windows. Для запуска программы на других операционных системах, необходимо скомпилировать ее с использованием соответствующих инструментов (см. раздел "Как сделать exe файл").

## Как сделать exe файл

Если вы хотите создать исполняемый файл (.exe) из исходного Python-скрипта, вы можете использовать инструменты, такие как PyInstaller. Следуйте инструкциям по использованию PyInstaller, чтобы создать .exe файл.
